# V1 Support Scope

This repository supports the open DQL adoption layer.

## Supported

- DQL syntax and parsing
- formatting and semantic analysis
- block testing and certification primitives
- project and registry primitives
- VS Code language support
- examples and starter templates

## Not supported here

- DuckCode notebook coworker
- natural-language generation
- source approvals
- MCP runtime
- product APIs and desktop/container packaging
