# Quickstart

This page now exists as a short compatibility entry point.

For the full onboarding flow, use the canonical guide:

- [Getting Started](./getting-started.md)

## Fastest path

```bash
npm install -g @duckcodeailabs/dql-cli
dql init my-dql-project --template ecommerce
cd my-dql-project
dql doctor
dql notebook
```

If you are not inside the project root, use `dql notebook ./my-dql-project` instead of a bare `dql notebook`.
