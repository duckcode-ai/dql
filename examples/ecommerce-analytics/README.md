# E-commerce Analytics

This example demonstrates a more polished DQL open-source evaluation path with browser notebook support and realistic commerce metrics.

## Run it

```bash
cd dql/examples/ecommerce-analytics
dql doctor
dql notebook
```

## Included assets

- `blocks/revenue_by_segment.dql`
- `blocks/repeat_rate.dql`
- `dashboards/revenue_command_center.dql`
- `notebooks/welcome.dqlnb`
